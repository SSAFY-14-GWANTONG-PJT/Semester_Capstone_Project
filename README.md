# 📚 Step-up Story: AI가 설계하는 동화 기반 유아 영어 교육 플래너 ✨

**AI 기반 상호작용형 동화로 아이 주도 학습을 실현하는 유아 영어 학습 관리 플랫폼**

---

## 💡 1. 프로젝트 개요 및 목표 (Service Overview)

| 구분 | 작성 내용 |
| :--- | :--- |
| **관통 PJT명** | AI가 추천하는 동화 기반 유아 영어 교육 플래너 |
| **서비스명** | **스텝업 스토리 (Step-up Story)** |
| **핵심 문제 해결** | 아이의 흥미 부족 및 부모의 수준별 교재 탐색 어려움을 **학습자 맞춤형 프로파일링**과 **AI 플래너**로 해결하여 체계적인 언어 성장을 유도. |
| **학습 철학** | 아동의 흥미와 학습 목표를 초기 진단하고, 이 데이터를 바탕으로 체계적인 주간/월간 학습 플래너를 AI가 설계하여 언어 능력이 선순환적으로 성장하도록 돕습니다. |

## 🎯 2. 주요 기능 및 강점 (Key Features)

| 구분 | 기능 요약 | 강점 |
| :--- | :--- | :--- |
| **1. AI 학습 플래너** | 초기 진단(레벨 테스트/관심사) 기반으로 **개인별 주간/월간 학습 계획(목표 어휘/문법)**을 자동 생성하고 대시보드를 통해 진도 및 성취도를 시각화. | **부모의 커리큘럼 설계 부담 해소** 및 체계적인 학습 로드맵 제시. |
| **2. 맞춤형 동화 생성** | 플래너의 **오늘의 목표 어휘 및 문법이 반드시 포함**되도록 동화 내용을 우선 추천·생성하며, 아이가 직접 **주제·등장인물·감정**을 선택하여 확장 가능. | 아이의 **흥미 기반**과 **학습 목표**를 동화에 유기적으로 결합. |
| **3. 인터랙티브 학습** | 음성 읽기(**TTS**) 및 따라 말하기(**STT**) 기능으로 **즉각적인 발음 피드백** 제공. | "읽고 → 듣고 → 말하는" 몰입형 학습 루프 완성. |
| **4. 어휘/문제 자동 생성** | 동화 본문의 핵심 어휘·문형을 **RAG 기반**으로 자동 추출하여 **카드화 및 복습(오답 노트)** 시스템과 연동. **문제 생성 Agent**가 동화 기반 맞춤형 문제를 출제. | **약점 보완** 중심의 선순환 학습 구조 및 반복 복습 용이성 확보. |

## 📝 4. 학습 선순환 흐름 (AI Learning Cycle)

1.  **진단/플래닝:** 연령, 수준, 관심사 진단 $\rightarrow$ **AI 학습 플래너** 자동 생성 (주간 목표 설정)
2.  **동화 읽기:** 오늘의 목표에 맞는 **맞춤형 동화 생성** 및 이미지 시각화
3.  **듣기/말하기:** 음성 듣고 따라 말하기 $\rightarrow$ **STT 발음 피드백**
4.  **문제 풀이:** 동화 기반의 **어휘/문법 문제** 풀이 및 결과 확인
5.  **피드백/반복:** 오답 노트 및 성취도 데이터 축적 $\rightarrow$ **다음 동화/플래너 생성에 반영**
