| 구분 | 내용 |
| :--- | :--- |

## 🎯 2. 주요 기능 (Key Features)

### 🪄 맞춤형 AI 동화 생성 & 멀티모달 학습
- **키워드 기반 창작:** 아이가 직접 주제, 등장인물, 장르를 선택하면 AI가 수준에 맞는 영어 동화를 즉석에서 집필합니다.
- **시각 및 청각 자극:** 동화 내용에 맞춘 **AI 삽화 자동 생성**과 **gTTS 음성 듣기** 기능으로 다감각 학습 환경을 제공합니다.

### 🎮 게이미피케이션 학습 시스템
- **동적 퀴즈:** 읽은 내용을 바탕으로 AI가 생성한 퀴즈를 풀며 어휘와 독해력을 점검합니다.
- **경험치(XP) 및 레벨업:** 학습 완료 시 경험치를 획득하며, 레벨업($100 + (n-1) \times 20$ 공식 적용)을 통해 지속적인 학습 동기를 부여합니다.

### 🌐 커뮤니티 & 나의 서재
- **공유 및 소통:** 직접 만든 이야기를 공유하고, **'좋아요'** 기능을 통해 친구들의 이야기를 추천하거나 모아볼 수 있습니다.
- **학습 관리:** 개인 서재에서 학습 현황(읽은 권수, 성장 포인트)과 동화 상태(공유/비공개)를 한눈에 관리합니다.

### 🔒 안전한 개인정보 관리
- **보안 강화:** 프로필 수정 시 **비밀번호 재인증 모달**을 통해 소중한 계정 정보를 안전하게 보호합니다.

## ⚙️ 3. 기술 스택 (Technical Stack)

### **Frontend**
- **Vue.js 3 (Vite):** 컴포넌트 기반의 빠르고 반응적인 UI 구현.
- **Pinia:** 학습 데이터(XP, Level, Token)의 상태를 효율적으로 관리하고 유지(`Persist`).
- **Axios:** 인터셉터를 활용한 보안 인증 및 백엔드 API 통신.

### **Backend**
- **Django (DRF):** RESTful API 설계 및 유저/콘텐츠 데이터 관리.
- **JWT Authentication:** 안정적인 사용자 인증 및 보안 세션 유지.
- **PostgreSQL / SQLite:** 정규화된 DB 설계를 통한 데이터 무결성 확보.

### **AI & Multimodal**
- **LangChain / Google Gemini:** 수준별 맞춤 동화 텍스트 및 퀴즈 생성.
- **OpenAI DALL-E:** 이야기 맥락에 어울리는 고품질 삽화 생성.
- **gTTS:** 자연스러운 음성 읽기 서비스 구현.

## 📝 4. 핵심 학습 사이클 (Learning Cycle)

1. **Create:** 아이의 관심 키워드로 **세상에 하나뿐인 영어 동화** 생성
2. **Read & Listen:** 눈으로 읽고 귀로 들으며 영어 문장과 표현에 노출
3. **Challenge:** 동화 기반 퀴즈를 풀고 **경험치 획득**
4. **Grow:** 획득한 포인트로 **레벨업**하며 나만의 동화 서재 완성
5. **Share:** 커뮤니티에 공유하고 친구들과 **좋아요**를 주고받으며 소통
